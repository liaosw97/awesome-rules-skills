# 错误码列表

## 附3：错误码列表

### 一级宏观错误码

| 错误码 | 中文描述 | 说明 |
|--------|----------|------|
| 00000 | 一切 ok | 正确执行后的返回 |
| A0001 | 用户端错误 | 一级宏观错误码 |
| B0001 | 系统执行出错 | 一级宏观错误码 |
| C0001 | 调用第三方服务出错 | 一级宏观错误码 |

### 用户端错误（A 类）

#### 用户注册错误（A0100）
- A0100 用户注册错误（二级宏观）
- A0101 用户未同意隐私协议
- A0102 注册国家或地区受限
- A0110 用户名校验失败
- A0111 用户名已存在
- A0112 用户名包含敏感词
- A0113 用户名包含特殊字符
- A0120 密码校验失败
- A0121 密码长度不够
- A0122 密码强度不够
- A0130 校验码输入错误
- A0131 短信校验码输入错误
- A0132 邮件校验码输入错误
- A0133 语音校验码输入错误

#### 用户登录异常（A0200）
- A0200 用户登录异常（二级宏观）

- A0203 用户账户已作废
- A0210 用户密码错误

#### 访问权限异常（A0300）
- A0300 访问权限异常（二级宏观）

- A0310 因访问对象隐私设置被拦截

- A0320 用户访问被拦截
- A0321 黑名单用户

- A0323 非法 IP 地址

#### 用户请求参数错误（A0400）
- A0400 用户请求参数错误（二级宏观）
- A0401 包含非法恶意跳转链接
- A0402 无效的用户输入
- A0410 请求必填参数为空
- A0420 请求参数值超出允许的范围
- A0430 用户输入内容非法
- A0440 用户操作异常

#### 用户请求服务异常（A0500）
- A0500 用户请求服务异常（二级宏观）
- A0501 请求次数超出限制
- A0502 请求并发数超出限制
- A0506 用户重复请求

#### 用户资源异常（A0600）
- A0600 用户资源异常（二级宏观）
- A0601 账户余额不足

#### 用户上传文件异常（A0700）
- A0700 用户上传文件异常（二级宏观）
- A0701 用户上传文件类型不匹配
- A0702 用户上传文件太大

#### 用户当前版本异常（A0800）
- A0800 用户当前版本异常（二级宏观）
- A0801 用户安装版本与系统不匹配
- A0802 用户安装版本过低

#### 用户隐私未授权（A0900）
- A0900 用户隐私未授权（二级宏观）
- A0902 用户摄像头未授权
