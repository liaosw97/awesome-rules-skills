# awesome-rules-skills

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

132+ 精选编码规则，专为 Claude Code 打造 — 66 个插件覆盖前端、后端、DevOps、AI、移动端、数据、安全等 32+ 技术领域。

## 安装

### 添加插件市场

```bash
claude plugin marketplace add liaosw97/awesome-rules-skills
```

### 安装单个插件

```bash
claude plugin install <plugin-name>@awesome-rules-skills
```

示例：

```bash
claude plugin install react@awesome-rules-skills
claude plugin install docker@awesome-rules-skills
```
